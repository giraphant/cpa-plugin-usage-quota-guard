Usage Quota Guard

这个页面本身是 CPA plugin resource；数据请求走需要 management key 的 /v0/management/plugins/usage-quota-guard/* API。

Management key 会保存在浏览器 localStorage，下次自动填入。插件本身无法读取 CPA 的 management key（CPA 隔离设计），所以需要你填一次。

添加下游 API key

同一个 key 只能添加一次，重复添加会被拒绝（请用「编辑」）。限额按 quota.period（weekly/monthly）计算。
